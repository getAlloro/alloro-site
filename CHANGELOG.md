# Alloro Site Changelog

All notable changes to Alloro Site are documented here.

## [0.0.1] - June 2026

### Terms Cancellation Notice Update

Updated the public Terms of Service so cancellation language matches Alloro's offboarding process while preserving the no-contract, cancel-any-time positioning.

**Key Changes:**
- Clarified that customers may request cancellation at any time, but cancellation is not immediate unless Alloro confirms an exception in writing.
- Added the 15-day notice rule for determining whether the current billing cycle or the next full cycle is the final paid period.
- Updated termination language so service access, data export, and infrastructure wind-down align with the final paid-period timeline.

**Commits:**
- `src/pages/TermsOfService.tsx` - amended Payment Terms and Termination copy for SOP-aligned cancellation expectations.
- `plans/06082026-terms-cancellation-notice-update/` - added the completed plan/spec artifact for the contract-facing copy update.
